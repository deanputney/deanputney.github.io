# Sunshine

A clean, modern blog theme for [Hexo](https://hexo.io/). Built for [Dean's Notes](https://deanputney.com/blog).

## Based on Minos

Sunshine is derived from the [Minos](https://github.com/ppoffice/hexo-theme-minos) theme by [ppoffice](https://github.com/ppoffice), maintained by [fltb](https://github.com/fltb/hexo-theme-minos). The original Minos repository was archived on Jun 4, 2022.

Sunshine strips Minos down to its essentials and rebuilds the design with:

- **Source Serif 4** body text + **DM Sans** headings
- Yellow (`#fb0`) link underlines that animate to highlights on hover
- Warm off-white background
- Wide image support via `[wide]` tag in markdown alt text
- Auto-generated captions from image alt text
- Removed: Bulma, jQuery, Font Awesome, moment.js
